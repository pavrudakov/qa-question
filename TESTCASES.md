Создание E2E-тестов для платформы с таск-трекером

Создание задачи
  1.	Нажать на кнопку "Создать задачу"
    •	Открывается окно "Редактирование задачи"
  2.	Заполнить необходимые поля
    •	Название
    •	Описание (необязательное)
    •	Проект
    •	Приоритет
    •	Статус
    •	Исполнитель
  3.	Нажать на кнопку "Создать"
    •	Задача должна появиться в "Списке задач"

Открытие карточки задачи
  1.	Нажать на необходимую задачу
    •	Открывается окно "Редактирование задачи" с полной информацией

Поиск задачи
  1.	Выбрать необходимую фильтрацию для поиска любым из способов
    •	Строка поиска
    •	Фильтр "Статус"
    •	Фильтр "Доска"
  2.	Найти необходимую задачу

Переход на доску проекта
  1.	Нажать на вкладку "Проекты"
    •	Открывается окно "Проекты"
  2.	Нажать на кнопку "Перейти к доске" у необходимого проекта
    •	Открывается окно проекта с существующими задачами
